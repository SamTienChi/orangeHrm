[Get] Job titles

Feature: Happy case
Scenario: #TC1 - Return default job titles when limit is 0
Given The system has job title
When user request job titles with :limit 0 , offset 0 , sortField=jt.jobTitle and sortOrder=ASC
Then response status must be 200
And The system returns all list of job titles with total
And the response should match JSON schema:
{
    "data": [
        {
            "id": number,
            "title": string,
            "description": string,
            "note": string,
            "jobSpecification": {
                "id": number | null,
                "filename": string | null,
                "fileType": string | null,
                "fileSize": string | null
            }
        }
    ],
    "meta": {
        "total": number
    },
    "rels": []
}
